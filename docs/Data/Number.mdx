Number adalah salah tipe data untuk angka, number dibagi menjadi 2, yaitu integer(bilangan bulat) dan decimal(pecahan)

## Integer

Cara menggunakan integer cukup menulis angka saja, contoh:

```js
console.log(250);
console.log(100);

// Pake variable
let nilai = 90
const umur = 21
```

## Decimal 

Menggunakan decimal di javascript tidak sama dengan menulis pecahan biasa, kita mingkin terbiasa menggunakan koma dalam pecahan, namun dalam javascript, decimal menggunakan titik

```js
console.log(2.5);
console.log(99.9);

// Pake variable
let nilai = 95.5 // berhasil
let nilaiUTS = 32,4 // error karna pake koma
```
