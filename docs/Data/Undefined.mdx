Undefined yang berarti tidak terdefinisi akan muncul ketika variable tidak [diinisialisasi](/referensi/kamus#initialization)

```js
let namaKosong

console.log(namaKosong) // undefined
```

Ingat!, undefined hanya muncul saat variable tidak diisi, bukan saat variable tidak ada, contoh kita ingin menampilkan variable harga

```js
console.log(harga) // error karna variable harga tidak ada
```
