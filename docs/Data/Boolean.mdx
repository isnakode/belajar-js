Boolean adalah tipe data yang berfungsi menyatakan benar atau salah, yang mana `true` berarti benar dan `false` berarti salah, contoh:

```js
console.log(true);
console.log(false);

// Pake variable
let apakahHujan = true
let apakahCantik = false
```

:::note[catatan]
Boolean juga bisa didapatkan dari operator [logika](/operator/logika) dan [perbandingan](/operator/perbandingan) yang akan dibahas di materi operator
:::
