# Instalasi

javascript sebenarnya bisa running di komputer hanya dengan [browser](https://www.google.com/search?q=apa+itu+browser+dan+contohnya), jadi cukup dengan chrome atau firefox tanpa software lain.

namun, agar belajar lebih mudah dan lancar, saya mewajibkan download node js dan visual studio code(vs code)

## Visual studio code (vs code)

visual studio code adalah salah satu [IDE](https://www.google.com/search?q=apa+itu+IDE+programming) populer saat ini, mudah digunakan, ringan dan tentunya support javascript.

langkah install vs code:

1. [download vs code](https://code.visualstudio.com/download)
2. setelah download selesai, double click pada installer yang sudah didownload
3. pastikan untuk mencentang "open with vs code" ![open with vs code](/img/open-with-vscode.png)

## Node JS

karna javascript hanya bisa running di browser, maka kita membutuhkan kekuatan dari node js untuk membuat javascript running diluar browser.

langkah install node js:

1. [download node js](https://nodejs.org/en/download)
2. setelah download selesai, double click pada installer yang sudah didownload
3. next hingga finish
4. buka cmd dan ketik `npm -v` maka akan muncul versi npm, lalu dilanjutkan dengan ketik `node -v` maka akan muncul versi node
<Tabs>
