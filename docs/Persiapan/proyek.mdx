# Membuat proyek

kita hanya butuh **satu** file untuk running javascript, namun agar file yang kita gunakan lebih tertata dan mudah dicari, kita akan membuat sebuah folder untuk menyimpan file-file tersebut, folder inilah nanti yang disebut proyek dan nama proyek tersebut adalah nama folder itu sendiri

ayo buat proyek pertama:

1. Buat folder di Document, namakan `dasar-js`
:::note[catatan]
tempat dan nama folder bebas, disamakan hanya agar menghilangkan kebingungan
:::
2. klik kanan pada folder dan pilih `open with vscode`
:::warning[peringatan]
windows 11 mungkin tidak ada pilihan `open with vscode` jadi kamu perlu klik show more option agar menu tampil
:::
3. vs code akan terbuka, perhatikan bahwa vscode menampilkan folder kita sebagai nama proyek
