## membuat file

untuk menulis kode, kita membutuhkan sebuah file javascript, untuk membuatnya, klik icon new file di explorer

![icon new file](/img/new-file.png)

dan namakan index.js

![icon new file](/img/indexjs.png)

perhatikan terdapat `.js` di akhiran file, itu menandakan file tersebut berisi javascript, klik file yang sudah dibuat dan ketikkan kode berikut
```js
console.log('hello world')
```

:::info
`console.log()` berfungsi menampilkan pesan ke dalam console
:::

## Run file

buka terminal dan ketikkan `node <nama file yang ingin dirun>`, karna nama file kita adalah index.js, maka kita ketikkan `node index.js` lalu enter

```bash
C:\Documents\dasar-js> node index.js
hello world
```

selamat!!! program pertamamu berhasil, coba ganti hello world dengan kata lain dan run ulang
