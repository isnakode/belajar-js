# Operator assignment

Operator assignment adalah operator yang berfungsi memasukkan data ke variable, beberapa operatornya adalah

| simbol | dibaca                | contoh     | sama dengan       |
|--------|-----------------------|------------|-------------------|
| =      | masukkan              | nilai = 5  | nilai = 5         |
| +=     | tambah lalu masukkan  | nilai += 5 | nilai = nilai + 5 |
| ++     | sama dengan `+= 1`    | nilai++    | nilai = nilai + 1 |
| -=     | kurangi lalu masukkan | nilai -= 5 | nilai = nilai - 5 |
| --     | sama dengan `-= 1`    | nilai--    | nilai = nilai - 1 |
| /=     | bagi lalu masukkan    | nilai /= 5 | nilai = nilai / 5 |
| *=     | kali lalu masukkan    | nilai *= 5 | nilai = nilai * 5 |

Contoh pemakaian:

```js
let nilai = 10
console.log(nilai) // 10
nilai = 15 // ini berhasil
console.log(nilai) // 15
```

Variable `nilai` diatas berhasil berubah karna kita masukkan langsung 15 ke dalam nilai, sekarang kita akan mencoba mengubahnya dengan cara menambah

```js
let nilai = 10
console.log(nilai) // 10
nilai += 5 // nilai(10) ditambah 5 menghasilkan 15, lalu 15 tadi masuk ke dalam nilai
console.log(nilai) // 15
```

Mohon untuk mencoba operator diatas satu persatu sendiri agar terbiasa

:::tip
Jika memang hanya ingin menambah 1 atau mengurangi 1, kita bisa menggunakan ++ / --
```
let angka = 5
angka--
console.log(angka) // 4
```
:::
