Berbeda dengan variable yang menggunakan `let` masih bisa berubah-ubah, constant sebaliknya akan tetap dengan nilainya selamanya, cara membuat constant cukup menggunakan `const` sebagai ganti let

```js
const gender = 'laki-laki'
const planetKita = 'bumi'
```

kita bisa menggunakan variable constant seperti variable biasa, hanya saja constant tidak bisa dirubah atau diganti dengan data baru

```js
const gender = 'perempuan'
console.log(gender) // ini berhasil

gender = 'laki-laki' // ini error karna constan tidak bisa diubah
```

:::tip
gunakan `let` jika variable ingin diubah, seperti: nilai, nama dan barang<br/>
gunakan `const` jika variable akan tetap, seperti: gender
:::
