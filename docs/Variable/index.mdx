# Variable

:::tip
Sebaiknya buat file baru khusus untuk belajar variable, seperti `coba-variable.js`, lalu run menggunakan `node coba-variable.js`
:::

Variable berfungsi sebagai tempat untuk menyimpan data-data, bisa diibaratkan seperti kotak yang berisi barang, atau kontak yang berisi nomor
