Variable tidak selamanya hanya menyimpan satu data, namun bisa diganti, bayangkan saja seperti sekolahmu yang **SD** akan berubah menjadi **SMA**, cara mengupdate/mengubah isi variable cukup dengan memanggil nama variable tersebut dan mengisinya dengan variable lain, contoh:

```js
let sekolah = 'SD'
console.log(sekolah);
sekolah = 'SMA' // update variable tidak perlu memakai let
console.log(sekolah);
```

jalankan kode diatas dan perhatikan bahwa tulisan yang pertama muncul adalah SD lalu SMA karna sudah diubah
