## Penamaan variable

Agar menjaga konsistensi, penamaaan variable memiliki beberapa aturan antara lain:

1. Penulisannya harus menggunakan [camel case](/Referensi/Kamus#camel-case)
2. Tidak boleh menggunakan symbol(%,*,@) dan nomor di awal
```js
let 5ribu // salah
let @nama // salah
```

## Harus satu

Tidak boleh ada 2 variable yang bernama sama

```js
let nama = 'herna' // ini berhasil
let nama = 'yati' // ini error karna variable nama sudah ada
```

Jika memang ingin mengubah isi, tidak perlu memakai let

```js
let nama = 'herna' // ini berhasil
nama = 'yati' // ini berhasil karna tidak membuat variable baru, hanya mengubah
```
