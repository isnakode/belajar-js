Setelah mengisi variable dengan data, kita bisa memakai/mengakses variable tersebut sebagai ganti data yang dimasukkan, contoh kita buat variable merk hp yang diisi dengan samsung
```js
let merkHp = 'samsung'
```

karna merkHp sudah berisi samsung, kita bisa menggunakannya tanpa harus langsung menulis langsung kata samsung

```js
// daripada gini
console.log('samsung')
// mending gini pake variable
console.log(merkHp)
```

:::tip[tugas]
cobalah membuat beberapa variable dan gunakan console.log() untuk menampilkan, lalu run
:::
