Tipe data

peringatan

Materi ini hanya pengenalan, materi untuk tiap tipe data akan dibahas lebih lanjut di materi data

Sebelum belajar variable kita sebaiknya mengetahui tipe data apa saja yang bisa dipakai di javascript, berikut beberapa tipe datanya:

String​

String adalah tipe data untuk tulisan, syntaxnya adalah menggunakan tanda kutip pada tulisan yang dibuat, contoh:

console.log('contoh pesan') // ini berhasil

Number​

Tipe data number dibagi 2 yaitu integer (bilangan bulat) dan decimal (pecahan), tipe data number tidak memerlukan tanda kutip, cukup angka saja

Integer​

console.log(200) // ini berhasil

Decimal​

Syntax pembuatan decimal di javascript berbeda dari pecahan biasa, biasanya kita menulis dengan koma seperti 4,5, tapi di javascript kita harus menggunakan titik 4.5

console.log(26.8) // ini berhasil

Boolean​

Boolean adalah tipe data yang berfungsi menunjukkan benar dan salah, boolean hanya memiliki dua data, yaitu true yang berarti benar dan false yang berarti salah

console.log(true) // ini berhasil
console.log(false) // ini juga berhasil

Array​

Array adalah salah satu tipe data yang berfungsi menyimpan kumpulan tipe data lain, array menyimpan datanya menggunakan index

console.log(['riski','ahmad','heri'])

Object​

Object adalah tipe data yang digunakan untuk menggambarkan benda yang ada di dunia nyata, seperti mobil, orang, pohon, dan lain-lain

console.log({
  nama: 'isnaini',
  hobi: 'coding',
  umur: 21,
})

Undefined​

Undefined muncul ketika variable tidak diinisialisasi
